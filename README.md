# ngMultiStepsContact

## Requirements

- [Node.js](https://nodejs.org/)
- [Gulp.js](http://gulpjs.com)
- [Bower](http://bower.io)

## Quickstart

```
git clone https://github.com/hic2h/ngMultiStepsContact.git
cd ngMultiStepsContact
npm install
bower install
```

Then run the serve `gulp` task:

```
gulp serve
```
